# Dead salmon brain

Python API for analysis A/B experiments

## Getting Started

### Build from source

```
1) Clone git repository
2) ./gradlew build
3) pip install python/
4) Add computing/build/libs/computing-*.*.*.jar to Spark jars/ folder
```
